# Del 20: Async IO

## Debug Mode
There are several ways to enable asyncio debug mode:
- Setting the PYTHONASYNCIODEBUG environment variable to 1.
- Using the Python Development Mode.
- Passing debug=True to asyncio.run().
- Calling loop.set_debug().
